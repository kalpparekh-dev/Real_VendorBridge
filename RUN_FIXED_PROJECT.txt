VendorBridge fixed project run steps

1) Backend setup
   cd server
   npm install

2) Edit server/.env
   Put your PostgreSQL password in DATABASE_URL.
   If the password contains @, write %40 instead.
   Example: Kalp1234@ -> Kalp1234%40

3) Database setup
   Make sure PostgreSQL is running and database vendorbridge exists.
   Then run:
   npx prisma generate
   npx prisma db push
   npm run prisma:seed
   npm run dev

4) Frontend setup in another terminal
   cd client
   npm install
   npm run dev

5) Open frontend
   http://localhost:5173
